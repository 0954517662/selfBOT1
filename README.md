## Installation for Termux

```sh
$ apt update
$ apt upgrade
$ apt install python
$ apt install git
$ git clone https://github.com/PHIEZUKE/selfBOT
$ cd selfBOT
$ python -m pip install -r requirements.txt
$ python SELFBOT.py
```

## Installation for VPS

```sh
$ git clone https://github.com/PHIEZUKE/selfBOT
$ cd selfBOT
$ python3 -m pip install -r requirements.txt
$ python3 SELFBOT.py
```
